# Birthday Web Application

## Description.
A web application that takes a ``user's birthday``, calculates the ``day of the week`` they were born and then depending on their gender outputs their ``Akan Name``. Akan names have a Ghanaian origin.

## BDD
| Inputs |  Description |
| :---         |          ---: |
| Century   | The century that the user was born in, ``eg 20th century``|
| Year     | The year of the century ``eg 98th year``   |
| Month     | The month of the year ``eg 7th month``     |
| Date     |  The day of the month ``eg 15th day`` |


| Outputs |  Description |
| :---         |          ---: |
| Day of birth  | The day of the week that the user was born ``eg Tuesday`` |
| Akan name    |  The corresponding Akan name for the user ``eg Kofi``    |
|     |      |
